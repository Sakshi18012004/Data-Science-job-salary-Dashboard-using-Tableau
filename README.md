# Data Science Job Salaries Dashboard

## Overview
This project is an interactive Tableau dashboard that analyzes Data Science job salaries across different experience levels, employment types, company sizes, job roles, and countries. The dashboard helps users understand global salary trends and workforce distribution in the data science industry.

## Dataset Description
The dataset contains Data Science job salary information including:

- Job Title
- Salary in USD
- Experience Level
- Employment Type
- Company Size
- Employee Residence
- Company Location

The data is used to analyze salary trends, global opportunities, and industry insights.

## Dashboard Features
- Salary analysis by experience level
- Employment type distribution
- Company size analysis
- Country-wise salary visualization
- Top employee residence analysis
- Interactive filters and visualizations

## Tools & Technologies Used
- Tableau
- Excel / CSV Dataset
- Data Visualization Techniques
